Help users to

Report a location using a map

Use an interactive map to allow users to select a location and retrieves the coordinates of the selected location.

Requirements

    The ability:

    • to select a location on an interactive map
    • to move the map to explore different areas
    • for users to confirm the accuracy of the pin's location
    • for grid references to be captured
    • for city, postcode/road information to be captured
